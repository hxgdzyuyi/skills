# 案例 006：开店前位置评估——保本线1600

**来源**：BV1REA8zYEnG
**标题**：保本线1600最低日销3000块 这样的位置能不能干？
**分类**：开店前评估 / 选址分析 / 竞品调研方法

---

## 基本信息

| 项目 | 数据 |
|-----|-----|
| 品类 | 链锅、油锅面、肉夹馍（多品类） |
| 位置 | 商场外围，科技馆/图书馆旁边，旁边是住宅小区 |
| 转让费 | 12万（对方开价），最低约10万 |
| 面积 | 55平 |
| 装修预算 | 5万 |
| 设备预算 | 3万 |
| 总开店预算 | 约31万 |
| 房租 | 11500元/月 |
| 人工 | 3人，约13500元/月 |
| 毛利率 | 55% |
| 每日保本线 | 1696元 |
| 老板背景 | 有餐饮经验（小吃5年），当地人 |

---

## 位置分析

### 优势
- 旁边大型商场（最好的商场），有一定客流
- 科技馆+图书馆（附近有文化设施，假期家长带孩子）
- 旁边住宅小区
- 商场内其他餐饮店日销普遍3000+，中午晚上有爆满情况

### 劣势（关键）
- **在动线尾部**：商场正门在大马路旁边（一号门），你的店在二号门侧
- 从图书馆/科技馆方向过来的客流，经过正面店铺后不会再往里走
- 最好的店都在正面动线上
- 你的位置是被主动线截流后的尾部

### 数据陷阱
- 用户来观察的时候是**暑假**（含假期）
- 周边科技馆、图书馆暑假人多，家长带孩子多
- 这是**旺季数据**，不代表平时水平
- 问题关键：**非假期的数据是多少？**

## 竞品调研情况

用户自述的周边数据（但存疑）：
- 商场内某面馆：日销3000+，中午约100单×18元=1800元，晚上类似
- 商场内另一家：每天3000多
- 沙县小吃：日销3000+
- 某转让的螺蛳粉店：高峰一天1万+，差的时候5000-6000
- 用户表示"没有低于3000的店"

**勇哥的质疑**：
> "你是自己数的，还是你只是觉得他门口有很多人？你作为一个餐饮人，不能说做满就叫爆满，你得数翻台率。"

---

## 勇哥的诊断

**位置问题**：
> "你的店在动线的尾部，我再说一遍，在动线的尾部。那边才是主动线，顺着这条路过来的人不会往你那边走的。"

**数据问题**：
> "你做的是暑假数据。如果非暑假是1000-2000，这个数据就值不起12万的转让费。"

**但如果数据是真的**：
> "如果你说就是都是3000以上，毫不犹豫拿！你盈亏别点才1600块钱，如果所有店都是3000以上，有啥不可以干的？"

---

## 勇哥的建议

### 建议一：验证非假期数据

> "你现在做的是暑假数据。你去看，如果非假期每家店也是3000以上，你对自己的话负责，你就去拿。"

**具体行动**：
- 在工作日（非假期）重新观察每家店
- **亲自数每家店的客流**，不是"感觉"而是实际数字
- 确认带孩子的客群占比（假期才有的客群vs常住客群）

### 建议二：压低转让费

> "你可以用旁边的在转的店来互相压价：你给螺蛳粉说这家只要9万，给这家说螺蛳粉只要7万，来回谈来回磨。"

- 目前有两家店在转让，互相竞争
- 磨下来的是利润

### 建议三：选品聚焦

> "做你之前说的那些粉面就好了，不要做这么多品类。"

---

## 核心方法论：开店前评估步骤

1. **确认动线**：你的店在主动线上还是尾部？
2. **验证竞品数据**：不是"感觉"，是亲自数
3. **区分旺季/淡季**：做的是什么时候的数据？
4. **计算保本线**：你需要每天卖多少才不亏？
5. **比较转让费合理性**：这个位置值这个钱吗？
6. **谈判策略**：利用竞争压价

---

## 关键教训

1. **动线比位置更重要**：就算在好商场旁边，在动线尾部也很难做
2. **旺季数据不可信**：必须看平时/工作日的数据做判断
3. **数数字不是感觉**：餐饮人必须养成数客流、数翻台率的习惯
4. **转让费要谈**：永远不是第一个报价就是最终价格
5. **保本线决定下限**：先算清楚每天至少卖多少才能不亏
