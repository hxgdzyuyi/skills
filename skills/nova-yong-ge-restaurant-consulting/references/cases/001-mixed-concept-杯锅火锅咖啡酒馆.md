# 案例 001：杯锅——混合概念文创园区店

**来源**：BV161AtzTERz
**标题**：投45万，耗时4个月打造的"破店"，结果月亏2万8，还要不要继续？
**分类**：混合概念 / 文创园区 / 新手创业

---

## 基本信息

| 项目 | 数据 |
|-----|-----|
| 店名 | 杯锅（火锅+咖啡+酒馆） |
| 城市 | 杭州 |
| 总投资 | 约45万 |
| 开店时长 | 装修4个月，开业约2个月 |
| 日销 | 约100元（极差） |
| 月销 | 约2000-3000元 |
| 房租 | 约2万元/月 |
| 人工 | 2人（老板+老婆），约1万/月 |
| 水电 | 约2000元/月 |
| 每月亏损 | 约2.86万元 |
| 合伙人 | 4人（老板、老婆、设计师、短视频人） |

---

## 核心问题

### 1. 文创园区位置陷阱
- 所在文创园区2024年底才开业，到咨询时仍未成熟
- 园区内商家稀少，整体人气不旺
- 这类园区前期需要漫长的"空白期"，成都东郊记忆起码熬了5年以上才火起来
- 两万块钱租金，要等不知道多少年

### 2. 品类混乱（三不像）
- 原本想开火锅 → 中途加入甜品/咖啡 → 又想开酒馆 → 最终三个不像
- "不是因为这个地方有客流有需求你来解决问题"，而是"觉得好玩"
- 火锅不像火锅、酒水不像酒水、咖啡不像咖啡

### 3. 合伙人问题
- 4个合伙人：全职老板、兼职老婆、设计师、短视频人
- 设计师和短视频人均未全力投入：该做视频的一直"在休息"
- 开店两个月，啥推广都没做，就等着客上门
- 开店像结婚看黄道吉日，拖延推广

### 4. 装修超期超支
- 两万多租金的店，装修装了4个月
- 装修费用约30多万（含设备搭建）
- 45万总投资基本全是装修和房租

---

## 勇哥的诊断

> "首先你们从一开始就没有想清楚。这样的文化创意园区前期都有很长时间的空白期，不会有客流。而且你们才是区区四个商家，你想这个地方活起来得等多久？两万块钱一个月的租金你能等多久？"

**三大核心问题**：
1. 文创园区前期无客流，等待时间不可控
2. 产品方向混乱，什么都想要什么都不精
3. 四个合伙人都不具备把这种店做起来的能力（营销/推广/运营）

---

## 勇哥的建议

**结论：关店**

> "直接关了。为什么？给你讲几个点……人要做自己和能力相匹配的事情。"

- 不要再拖延，越拖亏得越多
- 纯粹点的酒馆？勇哥反问："大家为什么到这来？酒馆卖点是啥？你们本身就没几张桌子能卖出多少钱？"
- 梦想可以有，但要和现实匹配

---

## 关键教训

1. **文创园区、工厂改造园区不适合新手**：这类位置需要大量资金和时间熬客流，不确定性极高
2. **装修时间=租金浪费**：两万租金装4个月，直接损失8万
3. **品类专一**：混合概念对品牌认知和运营能力要求极高，新手不适合
4. **合伙要有分工**：合伙人要各有能力，不能都是外行
5. **推广要从第一天开始**，不能等到"合适时机"

---

## 适用场景

当用户遇到以下情况时，参考此案例：
- 在文创园区/新开发商业区开店
- 店铺品类混乱（火锅+咖啡+酒馆等混搭）
- 合伙人多但各有拖延/不专注
- 开店了但推广迟迟没做
