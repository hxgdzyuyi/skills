# 案例 004：包工头跨行做餐饮——商场三楼火锅

**来源**：BV1PkNgzAEJb
**标题**：包工头跨行做餐饮 8个月亏50万 找不到原因
**分类**：跨行新手 / 商场选址 / 营销能力缺失

---

## 基本信息

| 项目 | 数据 |
|-----|-----|
| 品类 | 牛肉海鲜自选火锅（仿照福州某品牌） |
| 城市 | 江西上饶（市区） |
| 位置 | 市区商场三楼 |
| 总投资 | 约50万 |
| 开店时长 | 8个月 |
| 商场扣点 | 15% |
| 日销（春节月） | 约666元/天（全月不到2万） |
| 日销（最好月10月） | 约2967元/天（月销8.9万） |
| 平均日销 | 约1000元 |
| 人均消费 | 约70元 |
| 毛利率 | 约50% |
| 面积 | 285平 |
| 原房租 | 1900元/天（后降至约？） |
| 人工 | 开始13-14人，现4人，约14000元/月 |
| 水电 | 约3000元/月 |
| 每日亏损 | 约700元 |
| 老板背景 | 包工头，今年没干工程，第一次做餐饮 |

---

## 核心问题

### 1. 商场本身没有客流
- 商场同意降扣点至15%，说明**商场在留住商家**——这是商场自己客流差的信号
- 能压扣点的商场，往往是招商不满、客流不足的商场
- 三楼更是商场里最难做的楼层

### 2. 完全跨行，能力不匹配
- 包工头出身，以前是甲方乙方大口吃肉大口喝酒的关系
- 完全不懂餐饮营销：请了达人推广，但连达人都请不明白
- 没有营销能力、宣传能力、统筹能力

### 3. 产品照抄，逻辑不通
- 因为"吃过几次觉得好，产品简单没门槛"就模仿某福州品牌
- 没有想清楚：人家那个品牌可能靠加盟费赚钱，靠营销赚钱，产品本身未必盈利
- 你以为"味道好"就能复制成功，其实你只看到了表面

### 4. 财务计算
```
日销 1000元
毛利率（扣掉15%扣点后）≈ 1000 × 0.85 × 50% = 425元
日固定成本 = 房租/天 + 人工/天 + 水电/天
           ≈ （按降后房租约900元）+ 466元 + 100元 = 约1466元
每日亏损 ≈ 约700元
每月亏损 ≈ 2.1万元
```

---

## 勇哥的诊断

> "首先，商场给你15%扣点就说明一个问题——商场本身没有什么客流，他想留你，你走招商就招不起来了。"

> "你以前干工地的，和来这儿干餐饮完全不搭嘎。你在这样的商场做生意需要营销能力、宣传能力、统筹能力，这些东西你都不具备。"

> "你产品一味去模仿，有没有想过人家是做加盟的，靠营销的，钱不一定是靠产品赚的？"

**核心结论**：
1. 商场无流量（硬伤）
2. 本人能力不匹配（跨行+不懂营销）
3. 产品策略照抄不思考

---

## 勇哥的建议

### 选项一：找有能力的合伙人

> "你自己不适合，但租金已经谈得便宜了，那就找一个会营销、会运营的人来联合经营。"

- 找懂餐饮营销的人入伙
- 先把菜品调整好
- 重新做推广、做营销、建立曝光

### 选项二：彻底关店

> "如果找不到那种人，这个店就不用开了，那50多万就亏完了。"

---

## 关键教训

1. **商场店是高风险品类**：需要极强的营销和推广能力，新手勿碰
2. **商场扣点越低，往往说明商场越没有客流**：不要把低扣点当成优惠
3. **看别人火就去模仿是最危险的逻辑**：你看到的是表象，不是商业模型
4. **跨行创业要谨慎**：原行业成功的思维方式未必适用于餐饮
5. **能力>资金**：有50万没有能力，不如有20万有能力

---

## 适用场景

当用户遇到以下情况时，参考此案例：
- 从其他行业跨行做餐饮（建筑/制造/销售等）
- 开在商场里，生意不好
- 以"看别人做得好"为理由开店
- 不懂营销但开了需要营销支撑的店
