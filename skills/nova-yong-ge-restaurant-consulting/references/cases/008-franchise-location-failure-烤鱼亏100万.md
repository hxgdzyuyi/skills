# 案例 008：连续失败——两次加盟，总亏100万

**来源**：BV1SWAqzGETc
**标题**：倾家荡产亏100万 老板娘束手无策 问题究竟出在哪？（精简版）
**分类**：连续失败 / 位置无客流 / 傻瓜生意理论

---

## 基本信息

| 项目 | 数据 |
|-----|-----|
| 第一次 | 火锅加盟店，亏60多万，开了大半年 |
| 第二次 | 七花椒烤鱼（家门烤鱼品牌），亏约30万 |
| 总亏损 | 约90-100万（全部积蓄） |
| 城市 | 济南 |
| 当前店（烤鱼）位置 | 非商圈，周围500米内没有任何餐饮店 |
| 当前日销 | 约400元（近期），以前月均约1800/天 |
| 人工 | 4人（含老板），约1.5万/月 |
| 房租 | 6000/月 |
| 水电 | 2000/月 |
| 毛利率 | 约60% |
| 每日亏损 | 约500元 |
| 每月亏损 | 约1.5万 |
| 老板背景 | 以前做附占，全部积蓄亏完，第一次加盟也是通过网络被推荐 |

---

## 位置分析（致命问题）

### 完全错误的选址
- 周边500米内**没有任何一家餐饮店**
- 全是其他业态
- 楼上是自建房，每层住一户，居住密度极低
- 表面上看像是"城中心"，但实际上没有餐饮消费人群聚集
- 这个位置是"进城的道而已，根本就没有客流"

### 位置选择的来源
- 由"选址老师"（玄子老师）推荐
- 来源不明，可信度存疑

---

## 加盟陷阱分析

### 第一次（火锅加盟）
- 通过网络找到，感觉总部靠谱
- 亏了60万，大半年后关店

### 第二次（烤鱼加盟）
- 加盟费约6-7万
- 选址由加盟公司的"选址老师"负责
- 开业后找不到总部
- "连麦直播间，为什么找不到总部"——观众已心知肚明

**规律**：连续两次被假加盟或弱加盟坑害

---

## 经营历程

- 第1个月：日均约2000元（开业效应）
- 第2个月：急速下滑，只有几百
- 做了推广：请达人，生意又好了一个月
- 停推广后：又回到几百
- 花了两万学习运营课程
- 整改后好了一个月，又下滑
- 最近：日销只有400元

**规律**：每次推广能短暂拉起来，停了就回落 → 位置没有自然流量，推广是创造需求而非满足需求

---

## 勇哥的诊断

### 核心问题：位置是硬伤

> "你这其实不是什么城中心，你周围方圆500米没有任何一家餐饮店，全是其他业态，而且楼上全是自建房，人口居住密度并不高，你哪有这么多客流？"

> "你做的是推广好了，停了又下来，那是什么导致的？位置有味道，可能有，品类也有问题，综合在一起，这就是你为什么不挣钱。"

### 韭菜分析

> "你是一个妥妥的大韭菜！你花了六七万加盟这个品牌，我就问你在当地谁认识？价值在哪？"

- 连续两次付加盟费
- 两次都没有查过品牌特许经营资质
- 两次都没有做选址前的竞品调研

---

## 勇哥的建议

### 方向一：关店（最推荐）

当前每天亏500元，每月亏1.5万。位置是硬伤，无法改变。

> "从长远角度讲，关掉。"

### 方向二：换地方换品牌（如果不死心）

> "如果你不死心，我看你在拍视频，愿意出镜，想做。那换成现在广东很火的品类（鸡），但我先告诉你，这个店最终一定会倒。"

**核心逻辑**：
- 你会拍视频 + 好位置 + 刚需产品 = 有机会
- 但这个位置的问题无法用推广来填补

### 傻瓜生意理论

> "先找一个傻瓜生意。什么叫傻瓜生意？有客流有需求，我把产品做好，大家都能进来消费，这是95%的人适合的生意模型。比如你旁边有个写字楼，对面有个医院，你在这卖粉卖面，大家每天有这个需求，你到这来把东西做好，大家说好吃，我明天来后天来，这就是傻瓜生意。"

**一加一大于二**：
> "傻瓜生意 + 你自己的视频能力 = 1+1>2"

---

## 关键教训

1. **连续失败的根源**：从未做过基本的选址调研（不数竞品、不看客流来源、不查周边业态）
2. **推广不能替代位置**：推广可以临时拉客，但位置没有自然流量，推广一停就回落
3. **加盟必须查资质**：两次都没查，结果两次都踩坑
4. **先找位置，再考虑品类和推广**：位置是一切的基础
5. **不要创造需求，要满足需求**：在没有人的地方开餐饮，是在创造需求，大多数人做不到

---

## 适用场景

当用户遇到以下情况时，参考此案例：
- 同一个人多次开店多次失败
- 每次推广有效，停了就没人
- 怀疑自己选址有根本问题
- 周边没有其他餐饮店但自己在做餐饮
- 想了解什么是"傻瓜生意"
