# 勇哥问题库

格式：`编号, 阶段, 问题`

## 阶段一：开场了解（先搞清楚情况）

```
Q001, 开场, 遇到什么情况了？（开场问题，让对方先说）
Q002, 开场, 这个店是做什么的？卖什么的？
Q003, 开场, 这个店头多少钱（总投资）？
Q004, 开场, 开了多久了？
Q005, 开场, 是加盟店还是自营？
Q006, 开场, 加盟费给了多少钱？加盟费之外还给了公司多少钱（设备/物料/首批货）？
Q007, 开场, 你今天主要是想找我解决什么问题？
Q008, 开场, 你以前做过餐饮吗？（新手还是老手）
Q009, 开场, 你是干什么行业出身的？
```

## 阶段二：财务诊断（算清楚账）

```
Q010, 财务, 现在一天卖多少钱？
Q011, 财务, 一个月营业额是多少？
Q012, 财务, 毛利率多少？（食材成本占多少）
Q013, 财务, 房租多少钱一个月？多大面积？
Q014, 财务, 几个人在店上？
Q015, 财务, 人工多少钱一个月？（每个人分别多少）
Q016, 财务, 水电费多少钱一个月？
Q017, 财务, 有没有负债？（是自己的钱还是借的）
Q018, 财务, 现在每天亏多少钱？（根据以上数据计算）
Q019, 财务, 人均消费多少钱？（客单价）
Q020, 财务, 是商场扣点还是固定租金？扣点是多少？
Q021, 财务, 外卖平台的抽成是多少？
```

## 阶段三：环境勘察（360度看周围）

```
Q022, 环境, 来，360度严定转圈，我看看什么商圈（让用户描述周边）
Q023, 环境, 在哪个城市？市区还是郊区还是乡镇？
Q024, 环境, 店铺左边是什么？右边是什么？
Q025, 环境, 正对面是什么？（小区/写字楼/商场/学校/医院/景区）
Q026, 环境, 背后是什么？
Q027, 环境, 客流主要从哪个方向进来？主动线在哪里？
Q028, 环境, 这附近的竞品（同类或相似品类的店）一天卖多少钱？
Q029, 环境, 附近最近的大型商场/超市在哪里？多远？
Q030, 环境, 周边有没有学校？多少学生？大学还是中学？
Q031, 环境, 周边有没有医院？是三甲大医院吗？
Q032, 环境, 楼上是什么？住宅还是办公？入住率怎么样？
Q033, 环境, 整条街/整个商圈的生意怎么样？最好的店一天卖多少？
Q034, 环境, 这个位置是假期才有人还是平时也有？
Q035, 环境, 你来考察的时候是工作日还是周末/假期？
Q036, 环境, 你有没有亲自数过附近各家店的客流量？（不是"感觉上"的估计）
```

## 阶段四：能力评估

```
Q037, 能力, 你做过什么推广？（达人、短视频、团购、直播）
Q038, 能力, 有没有在做短视频或直播？账号情况如何？
Q039, 能力, 为什么选择这个品类？
Q040, 能力, 你有几个合伙人？他们分别是什么背景/职业？
Q041, 能力, 你们有没有人懂营销或短视频运营？
Q042, 能力, 这个产品是你自己研发的还是跟着别的品牌学的？
Q043, 能力, 你能不能搞定这个店需要的所有能力（产品、运营、推广）？
```

## 阶段五：品牌/加盟核查

```
Q044, 加盟, 怎么找到这个加盟品牌的？（网上搜的还是被推荐的）
Q045, 加盟, 有没有去实地看过总部或者门店？
Q046, 加盟, 有没有查过这个品牌的特许经营资质？
Q047, 加盟, 你打那个加盟电话是在网页上直接点的还是自己手动输入的？
Q048, 加盟, 找不到总部了吗？（当用户说联系不上公司时）
Q049, 加盟, 签没签合同？合同里有没有特许经营相关条款？
```

## 阶段六：产品与运营

```
Q050, 产品, 产品有没有收到顾客投诉？主要投诉什么？
Q051, 产品, 出餐速度怎么样？（堂食等待时间）
Q052, 产品, 用的是不是冷冻原料/预制菜？还是新鲜现做？
Q053, 运营, 开业以来做过哪些推广活动？
Q054, 运营, 美团/大众点评评分怎么样？有没有差评？主要差评是什么？
Q055, 运营, 有没有做过团购？
```

## 阶段七：决策问题

```
Q056, 决策, 你今天最想解决的核心问题是什么？
Q057, 决策, 如果继续开，你还能撑多久（资金还能支撑几个月）？
Q058, 决策, 如果关掉，你能接受吗？
Q059, 决策, 如果换品类，你有没有想好换什么？
Q060, 决策, 你旁边有没有在转让的店？转让费多少？
```
